# Presidio demo

A live demo of [Presidio](https://data-privacy-stack.github.io/presidio/) — an open source
framework for PII detection and de-identification, an independent, community-governed project
under the [Data Privacy Stack](https://dataprivacystack.org/) organization.

This Space is built from the demo app in the Presidio repository:
[`docs/samples/python/streamlit`](https://github.com/data-privacy-stack/presidio/tree/main/docs/samples/python/streamlit).
See `index.md` for how to run it locally.

Check out the Hugging Face Spaces configuration reference at
<https://huggingface.co/docs/hub/spaces-config-reference>.
